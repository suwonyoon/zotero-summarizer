## Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: High enough to read text clearly (at least 1280px wide for main screenshots)
- **Privacy**: Ensure no sensitive information is visible (API keys, personal documents, etc.)
- **Annotations**: Consider adding arrows or highlights to important UI elements
- **Consistency**: Use the same Zotero theme/style across all screenshots

## Tools for Screenshots

- **macOS**: ⌘+Shift+4 (select area), ⌘+Shift+5 (screenshot tool)
- **Windows**: Snipping Tool, Win+Shift+S
- **Linux**: gnome-screenshot, Spectacle (KDE), Flameshot
- **GIF recording**: LICEcap, ScreenToGif, Kap (macOS)
